# Number row characters
- Shift + 1: ! (Exclamation mark)
- Shift + 2: @ (At symbol)
- Shift + 3: # (Hash symbol)
- Shift + 4: $ (Dollar sign)
- Shift + 5: % (Percentage sign)
- Shift + 6: ^ (Caret)
- Shift + 7: & (Ampersand)
- Shift + 8: * (Asterisk)
- Shift + 9: ( (Left Parenthesis)
- Shift + 0: ) (Right Parenthesis)
